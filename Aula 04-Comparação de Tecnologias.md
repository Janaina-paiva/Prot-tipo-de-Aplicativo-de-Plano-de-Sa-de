# Aula 04: Comparação de Tecnologias (Refinamento de Prompt)

Nesta etapa, exploramos como o refinamento de um prompt transforma uma resposta genérica em uma análise técnica útil e estruturada.

## 1. Comparação Direta: Java vs. Python

**Prompt Refinado:** "Compare Java e Python em uma tabela com quatro colunas: paradigma de programação, desempenho, curva de aprendizado e principais aplicações em desenvolvimento de sistemas corporativos."

### Tabela Comparativa

| Critério | Java | Python |
| :--- | :--- | :--- |
| **Paradigma** | Orientado a Objetos (estrito), Multitarefa. | Multiparadigma (OO, Funcional, Imperativo). |
| **Desempenho** | **Alto** (Compilado para Bytecode, otimizado pela JVM). | **Moderado** (Linguagem interpretada). |
| **Curva de Aprendizado** | **Moderada** (Sintaxe verbosa e tipagem estática). | **Baixa** (Sintaxe limpa e intuitiva). |
| **Aplicações Corp.** | Sistemas bancários, Back-end de alta escala. | Ciência de Dados, IA, Automações e APIs. |

## 2. Análise Crítica
A utilização do formato de **tabela** facilita a análise por permitir a comparação visual imediata entre os pontos. Enquanto um texto corrido exige um esforço cognitivo maior para extrair dados, a tabela organiza as informações de forma que o usuário identifique as vantagens e desvantagens de cada tecnologia em segundos.
